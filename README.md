## Tecnologias

- pandas

## Como executar

### Criar ambiente virtual
    python -m venv venv

## Ativar ambiente

    Windows:
        venv\Scripts\activate

## Instalar dependências:
    pip install -r requirements.txt

## Executar projeto:
    python src/scraper.py
